Grafbase Extensions and Composite Schemas enable integrating non-GraphQL data sources like REST APIs and gRPC services into GraphQL Federation, addressing customization needs and the N+1 performance problem. It compares Grafbase's approach to Apollo and Cosmo using practical examples with public French government endpoints.
